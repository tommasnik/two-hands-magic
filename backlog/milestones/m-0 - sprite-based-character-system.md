## Description

Refaktor hry na genericky sprite-based system. Hra rotuje 5 characteru s realnymi sprity (stone-giant, plague-rat, ice-giant, crystal-spider, ember-wisp). Odstraneni veskereho proceduralniho renderovani a legacy hit detection. Endless mode s sekvencni rotaci enemy.
